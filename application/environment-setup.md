# Environment Setup

## Purpose

This document explains how the Customer Order Management System test application is set up locally for QA testing.

## Local Environment

The application is run locally using:

- macOS
- Python Flask
- MySQL Workbench
- MySQL database
- Terminal
- Web browser

## Setup Summary

1. Downloaded and extracted the application package.
2. Loaded the database schema into MySQL.
3. Loaded seed test data into MySQL.
4. Configured the `.env` file for database connection.
5. Started the Flask application from Terminal.
6. Opened the application in the browser.
7. Logged in using test credentials.
8. Performed initial smoke testing.
